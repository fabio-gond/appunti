Android Studio Last Release Updates

3.1 - Marzo 2018

  • Nuovo dexer, versione D8 (prima era DX) - il compilatore che converte il codice di programmazione .class in codice bytecode .dex diventa più veloce a compilare e crea file .dexleggermente meno pesanti. Interessante articolo sulla compilazione e il desugaring
  • Aggiornamento IntelliJ IDEA 2017.2 e 2017.3
  • Semplifica la finestra del Build
  • Aggiornamento Kotlin e aggiunta del supporto per il comando di lint ./gradlew lint
  • Miglioramenti del Layout Editor
  • Miglioramento del Code Editor quando si lavora con Room
  • SQL code completion
  • Emulatore android - aggiunta del Quick Boot: salva uno snapshot prima che chiudi l'emulatore

Quick Edit - Modifica Rapida

Modifica rapida del codice CSS associato ad un elemento HTML.

Cliccare col sinistro sul tag di un elemento, cliccare col destro e selezionare "Modifica Rapida" (CTRL+E). Se esiste, si aprirà il codice CSS associato a quell'elemento.
