Dreamweaver Tips and Tricks

Quick Edit - Modifica Rapida

Modifica rapida del codice CSS associato ad un elemento HTML.

Cliccare col sinistro sul tag di un elemento, cliccare col destro e selezionare "Modifica Rapida" (CTRL+E). Se esiste, si aprirà il codice CSS associato a quell'elemento.

Emmet

Serve per velocizzare un sacco la scrittura del codice.

E' già incluso in Dreamweaver CC. Se per esempio scrivo div e poi clicco il tasto tab si creaerà da solo il div con tag di apertura e chiusura

Multi Cursor

Posso scrivere simultaneamente su più righe moltiplicando il cursore.

Se devo scrivere nella stessa posizione su più righe consecutive devo usare ALT + click sulla prima riga dove voglio scrivere + spostare cursore in basso attraverso le varie righe.

Se devo scrivere in righe diverse uso CMD + click nei vari punti in cui voglio inserire del testo.

Auto-Formattazione del codice

Cliccando su Modifica -> Codice -> Applica Formattazione Origine oppure sul tasto Formatta codice di origine -> Applica Formattazione Originale mi formatta il codice in automatico.

Duplicare velocemente codice

Con CTRL + D duplico istantaneamente la riga di testo in cui mi trovo oppure il testo che è selezionato

Aggiustamento dei Numeri

Posso modificare velocemente un numero posizionando il cursore sul numero e poi usando la combinazione CTRL + ALT + Freccia SU per aumentarlo leggermente o CTRL + ALT + Freccia GIU per diminuirlo.

Includere parti di codice in un nuovo tag

Selezionare la parte di codice da racchiudere con il nuovo tag e cliccare CTRL + T.

Selezionare un intero tag rapidamente

Con la finestra DOM di Dreamweaver posso selezionare velocemente il tag in cui si trova al momento il mio cursore, i suoi parent o child, e altri tag.

Snippets

Il pannello Snippet di Dreamweaver contiene tanti codici già pronti in diversi linguaggi, HTML, CSS, Javascript, PHP, SASS etc...

Contiene per esempio lo snippet CSS per il reset, un semplice header HTML e così via.

Posso anche affibiare una Shortcut agli snippet che uso maggiormente.

Inserire testo Lorem Ipsum

Tra gli snippet HTML c'è anche il testo Lorem Ipsum che è anche utilizzabile con la cominazione lorem + TAB
